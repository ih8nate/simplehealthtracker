# simplehealthtracker
Health tracker to track weight and medications (good for glp-1 users) Weight tracker (with line graph to show change over time) and medication tracker on bottom. Could add more like exercise and macros just lmk

Weight Graph features 📈

Line graph with gradient fill that updates live as you add entries
Stats bar showing current weight, starting weight, total change, and entry count
▲ / ▼ change badges on each log entry showing gain/loss since previous entry
Collapsible entry log so the graph stays clean
Duplicate date protection — it'll ask before replacing

Medication Tracker features 💊

Date, medication name, and dose fields
Dose is optional — defaults to — if left blank
Newest entries show at the top
Hover any entry to reveal the delete button

Shared features 🌗

Friendly rounded Nunito font
Light/dark toggle that remembers your preference
Graph redraws correctly when you switch themes or resize the window
